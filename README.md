# Infinite-tik-tak-toe
Jogo da velha, sem velha. Sempre que atinge 3 peças iguais no tabuleiro a próxima remove a mais antiga e assim por diante até que alguém vença o jogo. O jogo tem um sistema de IA com 4 dificuldades: "Fácil", "Médio", "Difícil" e o modo "Vingança da Véia" -- Esse jogo foi desenvolvido para o estudo prático de minimax e poda alpha-beta.
